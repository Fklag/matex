Exercice 2

Les nombres de la forme Fn=22n+1F_n = 2^{2^n}+1Fn​=22n+1 où n∈Nn \in \mathbb{N}n∈N, sont-ils tous premiers ?

Réponse :

Premiers exemples :

  • F0=220+1=21+1=3F_0 = 2^{2^0}+1 = 2^1+1 = 3F0​=220+1=21+1=3, est premier.
  • F1=221+1=22+1=5F_1 = 2^{2^1}+1 = 2^2+1 = 5F1​=221+1=22+1=5, est premier.
  • F2=222+1=24+1=17F_2 = 2^{2^2}+1 = 2^4+1 = 17F2​=222+1=24+1=17, est premier.
  • F3=223+1=28+1=257F_3 = 2^{2^3}+1 = 2^8+1 = 257F3​=223+1=28+1=257, est premier.

Pour vérifier que 257257257 est premier, avec notre méthode, on utilise 257≈16\sqrt{257}\approx 16257​≈16 divisions. Une amélioration serait juste de vérifier que 257257257 n'est divisible par aucun nombre premier inférieur ou égal à 161616, à savoir : [2,3,5,7,11,13][2, 3, 5, 7, 11, 13][2,3,5,7,11,13] ; ainsi en 666 divisions on peut prouver que 257257257 est premier.

  • F4=224+1=216+1=65 537F_4 = 2^{2^4}+1 = 2^{16}+1 = 65\,537F4​=224+1=216+1=65537, est premier.

Pour vérifier que 65 53765\,53765537 est premier, avec notre méthode, on utilise 65 357≈256\sqrt{65\,357}\approx 25665357​≈256 divisions. Une amélioration serait juste de vérifier que 65 53765\,53765537 n'est divisible par aucun nombre premier inférieur ou égal à 256256256, à savoir une liste de 545454 nombres [2,3,5,7,...,239,241,251][2, 3, 5, 7, ..., 239, 241, 251][2,3,5,7,...,239,241,251]. Ce qui peut se faire sans calculatrice, et a été prouvé par Pierre de Fermat.

FnF_nFn​ : Les nombres de Fermat
Ces nombres doivent leur nom à Pierre de Fermat, qui émit la conjecture en 1640 que tous ces nombres étaient premiers.

  • F5=225+1=232+1=4 294 967 297F_5 = 2^{2^5}+1 = 2^{32}+1 = 4\,294\,967\,297F5​=225+1=232+1=4294967297. Ce nombre n'est divisible par aucun nombre premier inférieurs à 256256256 ; cependant il faudrait tester jusqu'à 65 53665\,53665536 pour prouver sa primalité avec notre méthode. En 1732, Euler utilise une autre méthode et trouve que 641641641 est un facteur de F5F_5F5​.

Aujourd'hui l'outil informatique permet une réponse rapide :

def is_prime(n):
    """ Test de primalité
    Version améliorée ; O(√(n))
    """
    if n < 2:
        return False
    for d in range(2, n):
        if n % d == 0:
            return False
        if d * d > n:
            return True
    return True

def F(n):
    """Renvoie le nombre de Fermat F_n"""
    return 2**(2**n) + 1

def vérifie_conjecture():
    """Vérifie la primalité des nombres de Fermat.
    S'arrête au premier nombre composé."""
    n = 0
    while True:
        Fn = F(n) # le nième nombre de Fermat
        if is_prime(Fn):
            print("F"+str(n)+" =", Fn, ", est premier.")
        else:
            print("F"+str(n)+" =", Fn, ", est composé.")
            print("Conjecture fausse.")
            return
        n = n + 1

def plus_petit_diviseur_premier(n):
    """Renvoie le plus petit diviseur premier de n > 1"""
    assert n > 1
    for d in range(2, n+1):
        if n % d == 0:
            return d

Exercice 6 : Décomposition en facteurs premiers

La question est de faire un premier script qui donne une décomposition en facteurs premiers.

def factor(n):
    """Renvoie une liste, la décomposition en facteurs premiers de n.
    >>> factor(6)
    [2, 3]
    >>> factor(8)
    [2, 2, 2]
    """
    assert n > 0
    Factor = []
    p = 2
    while n > 1:
        if n % p == 0:
            n //= p
            Factor.append(p)
            while n % p == 0:
                n //= p
                Factor.append(p)
        p += 1
    return Factor

for n in range(7, 13):

def factor(n):
    # variante qui sort une liste de couples (p, e)
    assert n > 0
    Factor = []
    p = 2
    while n > 1:
        if n % p == 0:
            n //= p
            e = 1
            while n % p == 0:
                n //= p
                e += 1
            Factor.append((p, e))
        p += 1
    return Factor
